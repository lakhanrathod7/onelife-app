# OneLife

Collect memories, not regrets.

## About

OneLife is a social bucket list application that helps users:

* Create life goals
* Track progress
* Share achievements
* Inspire others
* Build meaningful memories

## Tech Stack

* Flutter
* Riverpod
* Supabase
* PostgreSQL
* GoRouter

## Features

* Authentication
* Bucket List Management
* Goal Completion Posts
* Social Feed
* Daily Progress Logs
* Streak Tracking
* Follow System

## Project Status

Currently in active development.
