# Git Workflow

## Branches

main

Production Ready Code

develop

Current Development

feature/*

Individual Features

Examples:

feature/auth

feature/profile

feature/feed

---

## Workflow

feature/auth
↓
develop
↓
main

---

## Common Commands

Create Branch

git checkout -b feature/auth

Switch Branch

git checkout develop

Check Status

git status

Add Files

git add .

Commit

git commit -m "message"

Push

git push
