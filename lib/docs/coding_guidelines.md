# Coding Guidelines

## Naming

Classes

PascalCase

Example:

AuthRepository

Variables

camelCase

Example:

userName

Constants

camelCase

Example:

appName

---

## Rules

Do not hardcode colors.

Do not hardcode strings.

Use AppColors.

Use AppStrings.

---

## Widget Rules

Keep widgets small.

Maximum 200 lines preferred.

Extract reusable widgets.

---

## State Management

Use Riverpod.

Avoid global variables.

---

## Error Handling

Always catch exceptions.

Show meaningful messages.
